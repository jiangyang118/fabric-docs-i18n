# peer version

peer version 命令可以查看 peer 的版本信息。它会显示版本号、Commit SHA、Go 的版本、操作系统及架构和链码信息。例如：

```
peer:

Version: 2.1.0 Commit SHA: b78d79b Go version: go1.14.1 OS/Arch: linux/amd64 Chaincode:

Base Docker Label: org.hyperledger.fabric Docker Namespace: hyperledger

```

## 语法

peer version 命令没有参数。

## peer version ``` Print current version of the fabric peer server.

Usage:

peer version [flags]

Flags:
-h, --help

help for version

```
